# defi

Sistem pencarian bekerja dengan mengombinasikan Reciprocal Rank Fusion (RRF) pada Qdrant, model embedding Qwen3 Embedding 4B yang telah dikompresi melalui teknik kuantisasi, serta sebuah model reranker. Pada tahap awal, setiap dokumen dan query diubah menjadi vektor numerik menggunakan Qwen3 Embedding 4B versi terkuantisasi, sehingga proses representasi semantik menjadi lebih efisien dari sisi memori dan latensi tanpa kehilangan makna secara signifikan. Qdrant kemudian melakukan beberapa strategi pencarian, seperti pencarian berbasis vektor dan pencarian berbasis kata kunci (misalnya BM25), yang masing-masing menghasilkan daftar dokumen dengan peringkat tersendiri. Hasil-hasil ini digabungkan menggunakan metode RRF, di mana skor akhir dokumen ditentukan berdasarkan posisi peringkatnya di setiap hasil pencarian, sehingga dokumen yang konsisten muncul di peringkat atas akan lebih diutamakan. Selanjutnya, sejumlah dokumen teratas hasil RRF dikirim ke model reranker untuk dilakukan penilaian ulang secara lebih mendalam dengan mempertimbangkan hubungan semantik antara query dan dokumen secara kontekstual. Dengan kombinasi ini, sistem mampu mencapai keseimbangan antara efisiensi komputasi melalui embedding terkuantisasi, ketahanan terhadap variasi query melalui RRF, serta peningkatan presisi hasil akhir melalui reranking.
